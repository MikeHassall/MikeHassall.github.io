Square Breathing (javascript)

After a visit to the doctor to help with anxiety in the family, I learnt about the square breathing technique.

I am a visual person so I found that keeping track of the time and stage worked well using a box. I went for a colour scheme using wellness colours.

Note: If it is running too fast, please check if you have another copy open in another browser!
